## About Becwork

Becwork is a simple php framework.
Main web dir == public/ dir

## Becwork features

 - Included webpack
 - Included composer
 - Controll panel
 - Simple config system
 - Simple tests system
 - Crypt utils
 - Mysql utils
 - Cookie utils
 - Session utils


## Becwork required
 - php 7.2 or <
 - mysql5.6 or <
 - php-cli
 - php-curl


## Dependencies
